Lädt Inhalte aus Ihrem Bookyt Vermietsoftware Account über die API und stellt sie direkt in Wordpress dar. Voraussetzung ist die aktivierte API in Ihrem Bookyt Account.

== Description ==

**bookyt Vermietsoftware**
---------------------
Das #1 WordPress-Plug-in für die Bookyt Vermietsoftware

Damit Du das Plugin auf Deiner Wordpress Installation einsetzen kannst, benötigst Du einen Live-Account, den Du unter www.bookyt.de eröffnest.

**Aufruf**
---------------------
Interaktiv - Standort, Kategorie und Kalender

<blockquote><pre>[bookyt token="xyz" call="calendar" account="youraccount"]</pre></blockquote>

Hinweis: Um eine Station vorauszuwählen, ergänze den Shortcode mit z.B. station_id="1"

Liste der verfügbaren Gruppen

<blockquote><pre>[bookyt token="xyz" call="category" account="youraccount"]</pre></blockquote>

Aufruf für die Einbindung Ihres Newsboards

<blockquote><pre>[bookyt token="xyz" call="news" account="youraccount"]</pre></blockquote>

Standorte

<blockquote><pre>[bookyt token="xyz" call="locations" account="youraccount"]</pre></blockquote>
